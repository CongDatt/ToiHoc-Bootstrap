# Tôi học Bootstrap
## 1. Container vs Container-fluid 
- Class Container thì sẽ có 2 khoảng padding 2 bên trái phải
- còn class container-fluid thì full màng hình

## 2. Thêm class background và class màu của text vào element:
### background:
- bg-primary: xanh dương.
- bg-success: xanh lá.
- bg-danger: đỏ
- bg-waring: vàng
- bg-dark: đen
### text:
- màu tương tụ với background
- text-primary: xanh dương...

## 3. Một số từ viết tắt:
- m: margin
- p: padding
- t: top
- l: left
- r: right
- b: bottom
- Ví dụ: 
        m-1: margin 4px.
        mt-1: margin top 4px
